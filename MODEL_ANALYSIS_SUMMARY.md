### 주요 발견사항

#### 1. 낮은 R² 값 (약 0.01)

**관찰**:
- Random Forest: 0.004 ~ 0.010
- XGBoost: -0.005 ~ -0.011 (음수!)

**원인**:
1. **데이터 특성**: 계층화 데이터는 이미 K-means로 그룹화됨
2. **정신건강 복잡성**: 10개 센서만으로 개별 점수 예측 한계
3. **높은 개인차**: 동일한 센서 값에도 개인별 반응 차이

**해석**:
- 점수 회귀보다는 **분류** 또는 **생존 분석**이 적합
- 개별 예측보다는 **그룹 분류**가 목적

#### 2. Random Forest가 일관되게 우수

**이유**:
- 계층화 데이터의 비선형 패턴 포착
- 과적합 방지 능력
- 앙상블 효과

#### 3. MAE (평균 절대 오차)

| 종속변수 | RF MAE | 해석 |
|---------|--------|------|
| Anxiety | 0.92 | 평균 ±0.92점 오차 |
| Depression | 0.96 | 평균 ±0.96점 오차 |
| Stress | 0.81 | 평균 ±0.81점 오차 |

**의미**: 약 ±1점 내외의 예측 오차

## 🏥 생존 분석 결과

### Cox PH 모델 성능

#### Concordance Index (C-index)

| 종속변수 | 전체 변수 | 필터링 (p<0.05) | 개선도 | 선택된 변수 수 |
|---------|----------|----------------|--------|---------------|
| **Anxiety** | 0.6807 | 0.6838 | +0.0031 | - |
| **Depression** | 0.6971 | 0.6998 | +0.0027 | - |
| **Stress** | 0.6730 | **0.6839** | **+0.0109** | 3개 (30%) |

**C-index 해석**:
- 0.5: 무작위 예측
- 0.6~0.7: 보통 수준
- 0.7~0.8: 좋은 수준
- 0.8~0.9: 매우 좋음

**결과 평가**:
- 모든 모델이 **0.67~0.70 수준** → 보통~좋음
- 필터링 후 모두 개선 (특히 Stress +1.09%)

#### Stress 변수 선택 결과

**필터링 전**: 10개 변수
**필터링 후**: 3개 변수 (30% 선택률)

**선택된 변수 (p<0.05)**:
- 3개 주요 변수만으로도 성능 향상
- 모델 단순화 + 성능 개선 동시 달성

#### 통계적 유의성

**Likelihood Ratio Test**:
- Anxiety: p < 10⁻²² (극도로 유의)
- Depression: p < 10⁻²² (극도로 유의)
- Stress: p < 10⁻²⁵ (극도로 유의)

**결론**: 모든 모델이 통계적으로 매우 유의함

**회귀 모델** (R² 낮음):

- ✅ 그룹 간 차이 파악에는 유용

**생존 분석** (C-index 0.67~0.70):
- ✅ 사건 발생 예측에 적합
- ✅ 위험 요인 식별 가능
- ✅ 실무적 활용도 높음

**추천**: 정신건강 모니터링 시스템에서는 **생존 분석** 또는 **이진 분류** 활용

## 📈 실무 활용 가이드

### 시나리오별 모델 추천

#### 1. 조기 경보 시스템
**목표**: 정신건강 악화 조기 감지

**추천 모델**: **Cox PH (필터링 버전)**
```python
# p<0.05 변수만 사용
selected_features = ['body_temperature', 'total_sleep', 'hrv']

# Cox PH 모델
cph = CoxPHFitter()
cph.fit(data, duration_col='time', event_col='event')

# 위험 점수 계산
risk_score = cph.predict_partial_hazard(new_data)
```

**장점**:
- 실시간 위험도 모니터링
- 해석 가능한 위험 요인
- 적은 변수로 빠른 계산

---

#### 2. 그룹 분류
**목표**: 사용자를 위험군별로 분류

**추천 모델**: **Random Forest + K-means**
```python
# 계층화 데이터 사용
rf = RandomForestClassifier(n_estimators=100)
rf.fit(X_hierarchical, y_level)

# 3단계 분류 (낮음/중간/높음)
user_level = rf.predict(new_user_data)
```

**장점**:
- 간단한 3단계 분류
- 맞춤형 개입 전략
- 자원 효율적 배분

---

#### 3. 개별 점수 예측
**목표**: 정확한 점수 예측 (고급)

**추천 모델**: **시계열 모델 (LSTM, Transformer)**
```python
# 원본 long format 데이터 사용
# 시계열 패턴 고려
lstm = LSTM(input_size=18, hidden_size=64, num_layers=2)
prediction = lstm(sequence_data)
```

**장점**:
- 시간적 패턴 포착
