📖 극좌표 변환이란?

극좌표 변환은 카테시안 좌표계(x, y)의 점을 극좌표계(r, θ)로 변환하는 수학적 기법입니다.

  • r (반지름, radius): 원점으로부터의 거리 - 두 특성의 결합된 크기(magnitude)를 나타냅니다.
  • θ (각도, theta): x축으로부터의 각도 - 두 특성 간의 상대적 관계(방향)를 나타냅니다.
  • 장점: 원본 좌표계에서는 드러나지 않는 비선형 패턴과 특성 간 상호작용을 포착할 수 있습니다.
  • 활용: 생체 신호 분석, 이미지 처리, 로봇 공학, 컴퓨터 그래픽스 등에서 광범위하게 사용됩니다.

변환 공식

r = √(x² + y²)
θ = atan2(y, x)

atan2는 모든 사분면에서 올바른 각도를 계산하는 함수입니다 (-π ~ π)

🔢 atan2 함수란?

atan2(y, x)는 일반적인 역탄젠트 함수(atan)의 확장 버전으로, 두 개의 인자를 받아 모든 사분면에서 올바른 각도를 계산합니다.

왜 일반 atan이 아닌 atan2를 사용할까?

  • 일반 atan(y/x)의 문제점:
    • x = 0일 때 나눗셈 오류 발생 (0으로 나누기)
    • 결과 범위가 -π/2 ~ π/2로 제한되어 1, 4사분면만 구별 가능
    • 예: atan(1/1) = 45°, atan(-1/-1) = 45° → 1사분면과 3사분면을 구분 못함
  • atan2(y, x)의 장점:
    • x와 y의 부호를 모두 고려하여 정확한 사분면 판별
    • x = 0인 경우도 안전하게 처리 (90° 또는 -90° 반환)
    • 결과 범위: -π ~ π (모든 사분면 커버, -180° ~ 180°)

사분면별 처리 방식

사분면 x, y 부호 atan2(y, x) 범위 예시
1사분면 x > 0, y > 0 0 ~ π/2 (0° ~ 90°) atan2(1, 1) = 45°
2사분면 x < 0, y > 0 π/2 ~ π (90° ~ 180°) atan2(1, -1) = 135°
3사분면 x < 0, y < 0 -π ~ -π/2 (-180° ~ -90°) atan2(-1, -1) = -135°
4사분면 x > 0, y < 0 -π/2 ~ 0 (-90° ~ 0°) atan2(-1, 1) = -45°

특수한 경우 처리

  • x = 0, y > 0: atan2(y, 0) = π/2 (90°) - 정북
  • x = 0, y < 0: atan2(y, 0) = -π/2 (-90°) - 정남
  • x > 0, y = 0: atan2(0, x) = 0 (0°) - 정동
  • x < 0, y = 0: atan2(0, x) = π (180°) - 정서
  • x = 0, y = 0: atan2(0, 0) = 0 (구현에 따라 정의되지 않을 수도 있음)

실제 적용 예시

// JavaScript에서
Math.atan2(30, 50)    // → 0.54 rad (30.96°)  - 1사분면
Math.atan2(30, -50)   // → 2.60 rad (149.04°) - 2사분면
Math.atan2(-30, -50)  // → -2.60 rad (-149.04°) - 3사분면
Math.atan2(-30, 50)   // → -0.54 rad (-30.96°) - 4사분면

// Python에서
import numpy as np
np.arctan2(30, 50)    # → 0.54 rad (30.96°)
np.arctan2(30, -50)   # → 2.60 rad (149.04°)
np.arctan2(-30, -50)  # → -2.60 rad (-149.04°)
np.arctan2(-30, 50)   # → -0.54 rad (-30.96°)

🔍 극좌표 값의 의미

반지름 (r) - 전체적인 활동 수준

  • 높은 r 값: 두 특성의 값이 모두 높거나, 둘 중 하나가 매우 높음을 의미 → 높은 활동성 또는 변화량
  • 낮은 r 값: 두 특성의 값이 모두 낮음을 의미 → 낮은 활동성
  • 예시: (수면, 걸음수)에서 r이 크다면 충분한 수면과 활발한 활동을 나타냄

각도 (θ) - 특성 간 균형

  • θ ≈ 0° (0 rad): X 특성이 Y 특성보다 훨씬 높음 (X 편향)
  • θ ≈ 45° (π/4 rad): X와 Y 특성이 균형을 이룸 (이상적인 균형)
  • θ ≈ 90° (π/2 rad): Y 특성이 X 특성보다 훨씬 높음 (Y 편향)
  • 예시: (심박수, HRV)에서 θ가 45°에 가까우면 심혈관 건강이 좋은 상태를 시사

왜 극좌표 변환이 효과적인가?

  • 비선형 관계 포착: 원본 좌표계에서는 선형 모델이 포착하기 어려운 원형, 나선형 패턴을 명시적으로 표현
  • 크기-방향 분리: 전체적인 수준(r)과 균형 상태(θ)를 독립적으로 분석 가능
  • 특성 상호작용: 두 특성 간의 복잡한 상호작용을 r과 θ의 조합으로 단순화
  • 실제 성능: 본 프로젝트에서 C-index가 평균 27.6% 향상 (0.5361 → 0.6839)

⚙️ 구현 세부사항

Python 구현 예시

import numpy as np

def cartesian_to_polar(x, y):
    """
    카테시안 좌표를 극좌표로 변환

    Parameters:
    -----------
    x : float or array-like
        X 좌표값
    y : float or array-like
        Y 좌표값

    Returns:
    --------
    r : float or array-like
        반지름 (원점으로부터의 거리)
    theta : float or array-like
        각도 (라디안, -π ~ π)
    """
    r = np.sqrt(x**2 + y**2)
    theta = np.arctan2(y, x)
    return r, theta

# 사용 예시
x = 50  # total_sleep
y = 30  # walk
r, theta = cartesian_to_polar(x, y)
print(f"r = {r:.2f}, θ = {theta:.2f} rad ({np.degrees(theta):.2f}°)")
# 출력: r = 58.31, θ = 0.54 rad (30.96°)

주의사항

  • 정규화: 변환 전에 특성을 적절한 범위로 정규화하는 것이 중요합니다 (예: StandardScaler)
  • 음수 값: atan2 함수는 음수 값도 올바르게 처리하므로, 정규화된 데이터의 음수 값도 문제없습니다
  • 각도 범위: atan2는 -π ~ π 범위의 값을 반환합니다 (모든 사분면 커버)
  • 특성 선택: 의미있는 관계가 있는 특성 페어를 선택하는 것이 중요합니다
