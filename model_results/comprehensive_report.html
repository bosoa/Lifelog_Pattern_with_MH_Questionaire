🎯 문제 정의

  • 목표: 정신건강 고위험군 (≥7점) 조기 탐지
  • 핵심 도전: 클래스 불균형 (고위험군 9.9~17.3%)
  • 평가 지표: F1-Score, Recall (Accuracy는 불균형으로 인해 부적합)
  • 임상 요구사항: 높은 Recall (고위험 환자 놓치지 않기)

🧪 실험 방법론

방법 접근법 목표
Baseline XGBoost with scale_pos_weight 기본 성능 측정
SMOTE 합성 소수 샘플 생성 데이터 레벨 불균형 해소

Ensemble RandomForest + XGBoost 결합 모델 앙상블로 강건성 향상

💡 주요 발견

  • 낮은 F1-Score: 전반적인 분류 성능 개선 필요
  • ROC-AUC ~0.54: 랜덤 추측보다 약간 나은 수준
  • Stress가 상대적으로 양호: F1 0.213으로 가장 높음

💡 권장사항 및 향후 방향

🎯 즉시 적용 가능한 개선

  • Stress 모델 배포: 임계값 0.2로 설정하여 87% Recall 달성
  • 2단계 스크리닝: 1차(낮은 임계값) → 2차(정밀 평가)
  • 알림 시스템: 고위험 확률 0.2 이상 시 조기 경보

🔬 중기 개선 방향

  • 시계열 특징: 7일/14일 이동평균, 변화율, 추세
  • 도메인 특징: 수면 파편화, 활동 패턴 변화
  • 개인화 모델: 유저별 베이스라인 대비 변화 감지
  • 다양한 앙상블: Logistic + XGBoost + Neural Network

🚀 장기 연구 방향

  • LSTM/Transformer: 시계열 순서 패턴 학습
  • 준지도 학습: 미응답 데이터 활용
  • 다중 작업 학습: Anxiety-Depression-Stress 동시 예측
  • 설명가능 AI: 임상 의사결정 지원을 위한 해석성
  • 외부 검증: 다른 코호트 데이터셋으로 일반화 평가

📊 데이터 수집 개선

  • 설문 응답률 향상: 현재 8% → 목표 30%+
  • 빈 센서 해결: wakeup_time, bed_time, blood_pressure
  • 라벨 품질: 전문가 검증, 다중 평가자 일치도
  • 균형 샘플링: 고위험군 오버샘플링 전략
