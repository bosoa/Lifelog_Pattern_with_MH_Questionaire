## 🌐 5. 좌표 변환 분석 - 획기적 성과!

### 5.1 좌표 변환 개요

본 연구에서는 3가지 좌표계 변환을 적용하여 데이터의 잠재 패턴을 발견했습니다:

1. **극좌표(Polar) 변환**: 2D 페어링
2. **원통좌표(Cylindrical) 변환**: 3D 확장
3. **구면좌표(Spherical) 변환**: 3D 구형 표현

### 5.2 극좌표 변환 (Polar Coordinate)

#### 변환 공식
```
반지름 (r) = √(x² + y²)  → 두 특성의 결합 크기
각도 (θ) = atan2(y, x)   → 두 특성 간 상대적 관계
```

#### 변환 전략
- **페어링**: 연속된 특성들을 순서대로 페어로 묶음
- **특성 개수**: N개 원본 → N개 극좌표 (N/2 쌍)
- **차원 유지**: 정보 손실 없는 변환

#### 적용 데이터
- HRV-Walk, 체온-산소포화도, 스틱센서-피부온도
- 수면 데이터(Light-Deep-REM), 혈당-심박수

#### 장점
1. **비선형 관계 포착**: 카테시안 좌표에서 선형이 아닌 관계를 극좌표에서 선형화
2. **크기-방향 분리**: 데이터의 magnitude와 direction을 명시적으로 분리
3. **주기성 모델링**: 각도 정보로 순환 패턴 포착
4. **특성 공간 재구성**: 새로운 관점에서 패턴 발견

#### C-index 비교: 원본 vs 극좌표 🏆

| 타겟 | 원본 C-index | 극좌표 C-index | 개선도 | 개선율 |
|------|--------------|----------------|--------|--------|
| **Anxiety** | 0.533 | **0.700** | +0.166 | **+31.2%** 🥇 |
| **Depression** | 0.540 | **0.683** | +0.144 | **+26.6%** 🥈 |
| **Stress** | 0.535 | **0.669** | +0.134 | **+25.0%** 🥉 |

**평균 개선**: +0.148 (**+27.6%**)

### 5.3 원통좌표 변환 (Cylindrical Coordinate)

#### 변환 공식
```
ρ (rho) = √(x² + y²)     → 반지름 거리
φ (phi) = atan2(y, x)    → 방위각
z = z                     → 높이 (유지)
```

#### 특징
- **3D 확장**: 극좌표를 3차원으로 확장
- **Native 좌표 시각화**: 원통 공간에서의 데이터 분포 파악
- **Phenotype 분석**: 정신건강 표현형별 패턴 차이 시각화

#### 활용
- 3차원 특성 조합 분석
- 수면 패턴의 3단계(Light-Deep-REM) 동시 분석
- 시계열 패턴의 원통형 표현

### 5.4 구면좌표 변환 (Spherical Coordinate)

#### 변환 공식
```
r (radius) = √(x² + y² + z²)  → 원점으로부터의 거리
θ (theta) = arccos(z / r)      → 극각 (zenith angle)
φ (phi) = atan2(y, x)          → 방위각 (azimuthal angle)
```

#### 특징
- **완전한 3D 구형 표현**: 데이터를 구 표면에 투영
- **다차원 패턴**: 3개 이상의 특성 간 복잡한 관계 포착
- **Native + Phenotype 시각화**: 구형 공간에서의 클러스터링

#### 활용
- 복잡한 다변량 관계 모델링
- 생체 신호의 통합적 표현
- 정신건강 상태의 구형 공간 맵핑

### 5.5 획기적 발견

✨ **모든 타겟에서 일관된 대폭 성능 향상!**

- 원본 데이터: C-index 0.536 (랜덤 수준)
- 극좌표 변환: C-index 0.684 (우수 수준에 근접)
- **Anxiety에서 최대 효과**: 0.533 → 0.700 (31.2% 향상)
  - 극좌표 표현이 불안 예측에 특히 유용함을 시사
- 원본에서 포착 못한 **비선형 관계와 특성 간 상호작용**을 효과적으로 모델링

5. **체계적 실험**: 36개 조합 분석으로 최적 전략 도출 📊
6. **배포 가능 모델**: 타겟별 최적 모델 즉시 실전 적용 가능 🚀

### 9.2 연구 기여

- **방법론 혁신**: 
  - 다중 좌표계 변환(극좌표/원통좌표/구면좌표)의 정신건강 예측 효과성 입증
  - 지능형 특성 그룹화 방법 비교 및 최적화
  - 좌표계 × 그룹화 방법의 조합 최적화 프레임워크 제시
  
- **임상적 가치**: 
  - 고위험군 조기 탐지 시스템 구축 기반 마련
  - Anxiety 예측 C-index 0.684 달성 (임상적으로 유용한 수준)
  - 타겟별 맞춤형 예측 모델 제공
  
- **데이터 과학**: 
  - 클래스 불균형 문제 해결 방안 제시
  - 비선형 특성 관계 포착을 위한 좌표 변환 전략
  - 대규모 조합 실험을 통한 체계적 최적화

### 9.3 향후 전망

본 연구를 통해 **KLOSDOM 프로젝트**는:
- ✅ 정신건강 고위험군 조기 탐지 시스템 배포 가능
- ✅ 다중 좌표계 변환 기법의 생체 신호 분석 적용 검증 완료
- ✅ **C-index 0.68+ 달성** → 임상적으로 유용한 예측 시스템 구축
- 🎯 지속적 데이터 수집으로 C-index 0.70+ 목표

**최종 목표**: 원통좌표 + 지능형 그룹화로 **C-index 0.70+ 달성** → 실전 배포
