🎯 Concordance Index (C-index) 비교

C-index는 생존 분석 모델의 예측 성능을 나타내는 지표입니다. 0.5는 무작위 예측, 1.0은 완벽한 예측을 의미하며, 일반적으로 0.7 이상이면 우수한 모델로 평가됩니다.

🔄 구면좌표 변환 방법론

변환 원리

카테시안 좌표계(x, y, z)의 데이터를 구면좌표계(r, θ, φ)로 변환하여, 데이터의 크기(magnitude), 방위각(azimuthal), 극각(polar) 정보를 명시적으로 추출합니다.

수학적 변환 공식

r = √(x² + y² + z²)

→ 세 특성의 결합된 크기(magnitude)

θ = atan2(y, x)

→ xy 평면에서의 방위각(azimuthal angle)

φ = arccos(z / r)

→ z축으로부터의 극각(polar angle)

변환된 특성 트리플 (예시)

  • rem_sleep + walk + body_temperature → r, θ, φ (3개 특성)
  • oxygen_saturation + stick_sensor + hrv → r, θ, φ (3개 특성)
  • total_sleep + skin_temperature + stick_sensor → r, θ, φ (3개 특성)

※ 특성 수가 3의 배수가 아닌 경우, 남은 특성은 극좌표 변환 또는 원본 유지

🎯 결론 및 권고사항

주요 결론

  • 3차원 구면좌표의 효과: 3개 특성의 복합적 관계를 r, θ, φ로 표현하여 원본 데이터 대비 예측 성능 향상
  • 타겟별 최적 좌표계: Depression과 Stress는 구면좌표에서 우수한 성능, Anxiety는 극좌표가 더 효과적
  • 차원 선택의 중요성: 데이터 특성에 따라 2D(극좌표) 또는 3D(구면좌표) 변환의 효과가 다르게 나타남
  • 임상적 적용 가능성: 모든 타겟에서 적절(0.6-0.7) 또는 우수(>0.7)한 식별력 달성

향후 연구 방향

  • 최적 특성 그룹핑 전략 연구 (어떤 3개 특성을 묶을 것인가?)
  • 하이브리드 접근: 극좌표와 구면좌표를 혼합하여 사용
  • 원통좌표계(Cylindrical Coordinate) 등 다른 좌표계 탐색
  • 딥러닝 기반 자동 좌표계 선택 모델 개발
  • 4차원 이상의 고차원 데이터에 대한 확장 연구
